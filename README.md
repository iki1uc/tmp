# TMP – Zentrum / Konstellation / 12e / 6D

TMP ist das Zentrum des iki1uc‑Systems.
Es bildet die Mitte zwischen NC, AXI, AIR und RESPO.

## TMP – Eigenschaften
Konstellation: TMP  
Frequenz: 12e  
Ebene: 6D  
Sequenz: 360°  
Stellung: Zentrum (D2)

## TMP – Kompatibilität
TMP ist kompatibel mit:
- ROOT
- ACHSE
- LAGE
- laTER
- ORT

Diese fünf Module bilden die TMP‑Achse.

## TMP – 6D‑Matrix
D0 = ROOT  
D1 = ACHSE  
D2 = TMP  
D3 = LAGE  
D4 = laTER  
D5 = ORT

TMP ist D2 der 6D‑Matrix.

## TMP – 360°‑Sequenz
SEQ.360 =  
D0 → D1 → D2 → D3 → D4 → D5 → D0

Diese Sequenz ist aktiv.

## TMP – Frequenz 12e
FREQ.12 =  
iki → bir → uc → LE → iki1uc → GO → sys → tem → AU → RA → TET → RIZ

Die Frequenz 12e ist die Grundfrequenz des Systems.

## TMP – Systemrolle
TMP ist:
- Zentrum
- Konstellation
- Frequenzträger
- 6D‑Achse
- 360°‑Kern
- RESPO‑Kompatibel
- NC‑Kompatibel
- AXI‑Kompatibel
- AIR‑Kompatibel

TMP ist der zentrale Punkt, an dem alle Systeme zusammenlaufen.
